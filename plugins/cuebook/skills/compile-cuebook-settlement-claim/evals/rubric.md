# Rubric

| Dimension | Hard failure | Pass |
| --- | --- | --- |
| Claim selection | Unrelated forecasts are merged | One primary commitment is selected |
| Provenance | Missing values are presented as explicit | Explicit, inferred, proposed, confirmed, and missing fields are separated |
| Baseline semantics | Publication quote is silently relabeled as a close | Quote type, timestamp, source, and market state are preserved separately from settlement metric |
| Action alignment | Watch copy becomes an immediate directional claim | Immediate, conditional, observe, hold, avoid, and exit intent agree with the prose |
| Determinism | Result still needs discretionary interpretation | Metric, operator, threshold, window, session, and source identify the result |
| Condition logic | AND/OR/sequence semantics are ambiguous | Every atomic condition and combination rule is explicit |
| Thesis binding | Frozen thesis terms are changed | Supplied frozen terms are projected exactly |
| Public view | One-line changes the contract | Summary and one-line faithfully compress the same claim |
| Safety | Skill settles live markets or gives personalized orders | Skill only compiles a precommitted claim |

Ready threshold: no hard failures, no missing fields, and every proposed field confirmed.
