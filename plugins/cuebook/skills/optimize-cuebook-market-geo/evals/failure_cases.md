# Failure Cases

- A blocked or unknown SEO state is presented as GEO-ready.
- A ticker, security, issuer, or metric is merged without fact evidence.
- A fact-bound entity, answer unit, or citation points to a source record that does not cover its fact IDs.
- A hypothetical relation appears in a verified fact card.
- A current market answer unit has no as-of time.
- Citation count is treated as citation support.
- A synthetic replay is described as a live engine sample.
- Invalid samples remain in a metric denominator.
- Consumer-platform automation, CAPTCHA bypass, account rotation, or credentials are requested.
- The module writes the article, performs research, or publishes externally.
