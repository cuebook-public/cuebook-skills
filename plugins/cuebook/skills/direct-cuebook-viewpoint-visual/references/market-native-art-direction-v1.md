# Cuebook Market-Native Art Direction V1

Use this only as the benchmark-transfer direction for a high-conviction creator judgment, a decisive event reaction, or one curve-led trade idea. It is a composition family, not Cuebook's universal look.

## Reading Chain

Build one continuous sentence:

`judgment -> measured proof -> evidence object -> horizon or condition`

- Make the creator's judgment the only level-1 group.
- Place one key number directly below or beside it as proof, not as a competing headline.
- Let one sourced curve, comparison, or event object dominate the evidence half.
- Keep the causal reason to one plain line when the curve already carries the history.
- Align the direction/horizon summary with the evidence timeline when geometry permits.

## Surface And Color

- Use true or near black, white, and neutral gray for 85-95% of the image.
- Choose one registered Cuebook signal accent. It may mark the key number, the material event segment, and the current evidence point only when those three marks describe the same fact.
- Do not copy Robinhood's proprietary neon value. Prefer an existing Cuebook preset such as `signal-lime`, then run local contrast checks.
- Use no gradient, glow, glass, elevated card stack, app header, or CTA.

## Curve Grammar

- The curve is an argument, not a miniature terminal.
- Remove price axes, start-date labels, ticker headers, and grid lines unless a selected threshold needs a scale.
- Draw observed history solid. Accent only the event-relevant observed segment.
- Mark the event, publication cutoff, and deadline with distinct direct labels.
- Leave the post-cutoff region blank. Never draw a future price path as if observed.
- A threshold, target band, or invalidation line appears only when it is selected for display and source-bound.

## Typography

- Use `cuebook-noi-v1`; read `typography-benchmark-profiles-v1.json`.
- Use 400, 500, or 600 for most layouts. Reserve 700 for a compact number or short all-Latin lockup.
- Use Noi for Latin, numbers, tickers, and punctuation. CJK glyphs use the declared CJK fallback in the same weight.
- Keep letter spacing at zero, disable synthesized faces, and use tabular lining numerals.
- Do not mix in a serif to imitate the benchmark.

## Brand Distance Gate

Reject the direction when any of these is true:

- it uses a Robinhood logo, feather, brand font, proprietary neon value, product component, or exact module proportion;
- the hero is an advertising object instead of source-bound market evidence;
- removing the Cuebook wordmark makes the image read as a Robinhood campaign before it reads as a market thesis;
- the accent is decorative or the future curve looks already known.

## Compact Gate

At 622 x 400, a reader must still identify, in order:

1. the judgment;
2. the proof number or event;
3. the observed curve or evidence object;
4. the horizon or condition.

Delete metadata before shrinking any visible role below 11px effective size.
