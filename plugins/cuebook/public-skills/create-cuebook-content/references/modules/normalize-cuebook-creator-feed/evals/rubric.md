# Creator Feed Rubric

| Dimension | 5 means |
| --- | --- |
| Identity | Durable entity IDs and dated symbol aliases resolve without ambiguity |
| Provenance | Exact source revisions, locators, hashes, clusters, and access bases are preserved |
| Temporal integrity | Every active revision is available by the frozen cutoff and replayable |
| Semantic integrity | News, schedules, hypotheses, ideas, executions, and outcomes never collapse into one type |
| Rights and privacy | Reuse rights, confidentiality, consent, and private trade fields are explicit |
| Correction readiness | Supersession, retraction, and same-as edges append history without overwriting it |

`ready` requires every applicable dimension at 5. Use `conditional` for usable internal data with unresolved public rights or disclosures. Use `blocked` for identity conflict, temporal leakage, invalid lineage, or fabricated verification.
