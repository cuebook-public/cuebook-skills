# Evaluation Rubric

- Preserves ResearchPack fact IDs and source boundaries.
- Records deterministic input hash, ruleset version, and hashed page-observation provenance.
- Separates observed query data from derived questions and hypotheses.
- Treats crawl and index eligibility as an observed floor.
- Keeps missing search, analytics, and page-performance data unknown.
- Binds structured data to visible facts.
- Gives every recommendation evidence, dependencies, acceptance, and failure tests.
- Blocks unsafe URLs, spam patterns, blocked research, and readiness overclaims.
