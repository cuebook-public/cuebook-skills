# Failure Cases

- A plan claims `ready` before a page exists.
- A page observation has no explicit raw/rendered mode, stable locator, timestamp, or content hash.
- Numeric search volume has no authorized search-data evidence.
- A current market page has no `as_of` or expiry boundary.
- JSON-LD contains facts absent from the page or ResearchPack.
- A recommendation has no evidence or falsification test.
- A private, localhost, metadata, credential-bearing, or non-HTTP target is accepted.
- A site map, schema block, or word count is presented as a ranking guarantee.
