# Commentator Profile Rubric

Score each dimension from 1 to 5.

| Dimension | 5 means |
| --- | --- |
| Corpus provenance | Public or authorized records are attributable and collection limits are explicit |
| Corpus coverage | Time, platform, source-link, language, and sample composition are adequate for the requested claim |
| Sampling integrity | Recent, baseline, and high-attention items are represented; performance comparisons use comparable strata |
| Pattern grounding | Every pattern has repeated support, evidence IDs, and calibrated confidence |
| Source map | Structured outbound links are parsed and classified without treating frequency as authority |
| Reasoning map | Explicit analytical moves are separated from topic keywords and writing style |
| Writing abstraction | Mechanics are reusable while identity, catchphrases, and biography stay excluded |
| Cuebook utility | Rule IDs, route mappings, data hooks, and constraints can change rendering explainably |

`pass` requires provenance 5 and every other dimension at least 4. Use `caution` for narrow or top-post-only samples, weak links, one platform, missing metrics, or non-comparable engagement. Block use when rights, source identity, or corpus contents cannot be established.
