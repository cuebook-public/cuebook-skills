# Market Media Rendering Rubric

- **Evidence**: each section, post body, card, beat, and asset claim binds to fact IDs.
- **Lineage**: each package resolves to selected opportunities, its program item, and all input artifacts.
- **State integrity**: publication cannot be less strict than projection, research, or policy.
- **Route integrity**: full RouteV1 is retained and an abstention cannot produce a public package.
- **Media fit**: the package uses the medium's units rather than pasting the same prose everywhere.
- **Human language**: concrete actors and conditions replace template narration and engagement bait.
- **Policy**: channel, community, qualification, sponsorship, and current-rule checks are explicit.
- **Rights**: every public asset is original or has reusable rights.
- **Originality**: profile and format rules shape selection without copying a living creator.
- **Decision value**: the artifact makes the asset expression and horizon legible, then ends with the next catalyst or observable. Counterevidence and invalidation remain structured unless the requested format has a dedicated risk-boundary module.
