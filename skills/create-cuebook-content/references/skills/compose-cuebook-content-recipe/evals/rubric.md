# Content Recipe Rubric

| Dimension | 5 means |
| --- | --- |
| Ingredient integrity | Every selected record resolves, is cutoff-safe, active, authorized, and correctly typed |
| Subject coherence | One primary subject connects every asset or names the proxy bridge |
| Preparation clarity | Research depth, reasoning lenses, market data, countercase, and invalidation are explicit |
| Flavor control | Voice traits adjust expression without copying a living author or changing claim strength |
| Plating fit | Each channel uses a supported format, count, context, and media rule |
| Skill resolution | User choices, mandatory skills, versions, dependencies, and extensions agree with the catalog |
| Reproducibility | Recipe revision, catalog version, cutoff, inputs, and version pins can recreate the run |

A valid recipe has no unresolved ingredient, permission, temporal, format, catalog, or mandatory-skill failure. Automatic presets remain conditional until their concrete ingredients and opportunity are resolved.
