# Stable Recipe Failure Cases

| Case | Required behavior |
| --- | --- |
| Primary ingredient is not selected | Block ingredient-first execution |
| Opportunity anchor is absent or unselected | Block opportunity-first execution |
| Two news items share one event cluster | Merge or remove the duplicate |
| Narrative is treated as a verified fact | Keep it as a research hypothesis |
| Scheduled calendar item is written as completed | Keep the recipe pre-event |
| Trade idea is treated as a fill | Preserve idea-only state |
| Private or self-reported winner is selected for performance content | Remove or block until authorized and reconciled |
| X is paired with carousel format | Repair the channel/format pair |
| Owned web omits SEO | Add the required skill |
| GEO is selected without SEO | Add SEO or remove GEO |
| Required evidence skill is missing | Restore it; user controls cannot disable it |
| Recipe catalog version differs from workflow | Re-resolve and create a new recipe revision |
| Seeking Alpha is configured as a release candidate | Limit it to an internal outline |
