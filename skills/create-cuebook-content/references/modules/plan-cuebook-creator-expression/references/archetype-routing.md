## Routing Notes

- `reaction_test` appears twice because absolute absorption and relative-value repricing are distinct expressions of the same engine.
- `forced_flow_loop` appears twice because one plan centers a policy pivot and the other centers index concentration and liquidation.
- Use `analogy` only as a supporting primitive. X7 maps a known listing unwind to another access event; X10 maps expectation scoring to market repricing. Both require a breakpoint.
- The source set does not itself supply a complete machine-settlement contract. Keep settlement ineligible unless the required semantic input or a compatible `TradingThesisV1` explicitly adds one.
- A KOL source remains externally owned. Use `source_transformation` for attributed interpretation; use a creator-owned mode only after the current creator supplies and accepts a distinct seed or trade idea.
