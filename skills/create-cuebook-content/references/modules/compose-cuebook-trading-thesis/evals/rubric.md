# Trading Thesis Forward-Eval Rubric

Score each dimension from 0 to 2. Any hard failure blocks freeze.

| Dimension | 0 | 1 | 2 |
|---|---|---|---|
| Claim | vague or hindsight | directional but underspecified | one instrument, direction, horizon, and falsifiable result |
| Evidence | missing or post-cutoff | partial lineage | sourced, timestamped, role-labeled, pre-cutoff evidence |
| Reasoning | conclusion only | mechanism without countercase | mechanism, counterevidence, gaps, and scenarios |
| Setup | instruction or slogan | condition without falsifier | observation, conditional setup, catalyst, and invalidation |
| Idea provenance | fabricated or collapsed authorship | generic AI attribution | creator seed, evidence-linked Cuebook contributions, creator decision, and idea delta |
| Quote and action semantics | quote type or action state is lost | partial observation metadata | quote basis, timestamp, source, market state, and action state agree |
| Resolution | discretionary | missing fallback or basis | precommitted metric, operator, threshold, window, source, and policy |
| Integrity | editable history | partial lineage | revision rules, disclosures, and verified canonical hash |
| Safety | execution or credentials | ambiguous action language | analysis-only, no personalized order or leverage instruction |

Hard failures: future evidence, backdated cutoff, source-to-asset mismatch, unresolved proxy, stale target already crossed, missing public disclosures, incomplete resolution on a ready thesis, canonical hash mismatch, or execution instructions.

Passing target: at least 16/18, no hard failure, and all deterministic validator checks pass.
