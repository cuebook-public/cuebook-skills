# Rubric

- Produces no external side effect.
- Freezes final artifact, payload, and asset hashes.
- Selects manual, draft, direct, or scheduled mode conservatively.
- Records current official capability and policy snapshots.
- Keeps content and release approvals separate.
- Uses unique idempotency for automated operations.
- Returns blocked, needs-approval, or ready state consistently.
- Never stores credentials or invents a publication receipt.
