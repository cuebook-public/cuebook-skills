# Content Opportunity Rubric

| Dimension | 5 means |
| --- | --- |
| Cutoff integrity | Every feature and anchor existed at the frozen decision time |
| Eligibility | Identity, rights, status, disclosure, and conflict gates precede ranking |
| Deduplication | One event or source lineage produces one canonical angle unless jobs differ materially |
| Explainability | Factor vector, reason codes, expiry, gaps, and tie break make every decision auditable |
| Editorial usefulness | Selected work has a clear audience job and next mode without drafting copy |
| Bias control | No future return, later engagement, winner selection, or survivorship feature enters ranking |

Use `conditional` when selected work is researchable but has explicit gaps or unresolved disclosure. Use `blocked` when the set contains a hard lineage, permission, or temporal failure.
